تقرير تحليل مشروع TransGame
اللغات والتقنيات المستخدمة:
لغات البرمجة:

TypeScript: اللغة الرئيسية المستخدمة في كتابة الكود المصدري للتطبيق.
JavaScript: يتم استخدامها بشكل غير مباشر من خلال TypeScript.
مكتبات وأطر العمل:

React: إطار العمل الرئيسي لبناء واجهة المستخدم.
Vite: أداة بناء وتطوير سريعة للمشروع.
Tailwind CSS: إطار عمل لتنسيق وتصميم واجهة المستخدم.
Capacitor: منصة لتحويل تطبيق الويب إلى تطبيق أصلي يعمل على الأجهزة المحمولة.
أدوات التطوير:

ESLint: أداة لفحص جودة الكود واكتشاف الأخطاء.
PostCSS: أداة لمعالجة وتحويل CSS.
Autoprefixer: إضافة تلقائية لبادئات المتصفحات إلى CSS.
Vite PWA: مكون إضافي لتحويل التطبيق إلى تطبيق ويب تقدمي (PWA).
مكتبات واجهة المستخدم:

Lucide React: مكتبة للأيقونات.
هيكل المشروع:
الملفات الرئيسية:

src/App.tsx: المكون الرئيسي للتطبيق.
src/main.tsx: نقطة الدخول الرئيسية للتطبيق.
capacitor.config.ts: إعدادات Capacitor.
vite.config.ts: إعدادات Vite.
هيكل المكونات:

src/components/: يحتوي على مكونات واجهة المستخدم مثل GameBoard و TargetWords.
src/hooks/: يحتوي على Hooks مخصصة لإدارة حالة اللعبة والمنطق.
src/types/: يحتوي على تعريفات الأنواع المستخدمة في التطبيق.
src/data/: يحتوي على بيانات الكلمات المستخدمة في اللعبة.
بيانات اللعبة:

src/data/words.ts و src/data/words1.ts: يحتويان على قاموس الكلمات الإنجليزية مع ترجمتها العربية.
كيفية عمل الأوامر:
npm run dev:

يقوم بتشغيل خادم التطوير المحلي باستخدام Vite.
يتيح معاينة التطبيق مباشرة في المتصفح أثناء التطوير.
يدعم Hot Module Replacement (HMR) لتحديث التغييرات فوراً دون إعادة تحميل الصفحة.
npm run build:

يقوم ببناء التطبيق للإنتاج.
يحول ملفات TypeScript و React إلى JavaScript محسن.
ينشئ مجلد dist يحتوي على جميع الملفات اللازمة لتشغيل التطبيق.
يضمن توافق الملفات مع المتصفحات المختلفة.
npx cap sync android:

يقوم بمزامنة ملفات الويب مع مشروع Android.
ينقل الملفات من مجلد dist إلى مشروع Android.
يضمن تحديث جميع التبعيات والإعدادات اللازمة لتشغيل التطبيق على Android.
npx cap open android:

يفتح مشروع Android في Android Studio.
يتيح بناء وتشغيل التطبيق على جهاز Android أو محاكي.
يوفر وصولاً كاملاً إلى أدوات تطوير Android.
